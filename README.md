# Task Management System

A full-stack task management application with drag-and-drop functionality, built with the MERN stack.

#Features
Authentication
- User registration and login
- JWT token-based authentication
- Protected routes
- Profile management

 Boards
- Create, read, update, and delete boards
- View all boards in dashboard
- Each board contains tasks organized by status

 Tasks
- Create tasks with title, description, and due date
- Drag and drop tasks between columns (Todo, Doing, Done)
- Edit and delete tasks
- Due date tracking with overdue indicators
- Task status management

# User Profile
- Update profile information
- Change password with current password verification
- View account statistics (total tasks, completion rate, etc.)
- Recent activity tracking
- Theme preferences (Light/Dark mode)
- Notification settings
- Account deletion

# Tech Stack

# Frontend

- Tailwind CSS
- React Router DOM
- React Beautiful DND
- Axios
- React Hot Toast
- React Icons
- Date-fns

# Backend
- Node.js
- Express.js
- MongoDB Atlas
